# Python ATM System

A beginner-level ATM simulation project built in Python using concepts learned during Day 1–15 of my Python learning journey.

## Features

* User login using username
* PIN verification system
* Check account balance
* Withdraw money
* Deposit money
* Multiple user accounts
* Conditional logic using `if`, `elif`, and `else`
* Interactive terminal-based interface

---

## Concepts Used

This project helped me practice:

* Variables
* Data types
* Type casting (`int()`)
* User input with `input()`
* Arithmetic operators
* Conditional statements
* String formatting
* Program flow logic
* Debugging and error tracing

## What I Learned

While building this project, I faced multiple logic and debugging issues such as:

* String vs integer comparison problems
* Managing program flow
* Handling user-specific balances
* Repeated input handling
* Understanding how programs store and use current state

I debugged most of the issues by testing outputs step-by-step and improving the logic iteratively.
